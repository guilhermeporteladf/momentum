# Momentum Landing Page

A premium, no-scroll event landing page for Momentum built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
